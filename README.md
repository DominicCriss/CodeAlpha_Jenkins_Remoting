# CodeAlpha - Jenkins Remoting

## Overview

This project demonstrates the setup of a Jenkins CI pipeline with a Windows Agent using Jenkins Remoting (WebSocket).

## Features

- Installed Jenkins on Windows
- Configured a Windows Agent
- Connected the agent using WebSocket
- Created a declarative Jenkins Pipeline
- Executed builds remotely

## Technologies Used

- Jenkins 2.555.3
- Java 21
- Windows 11

## Pipeline

The pipeline:
1. Connects to the Windows Agent.
2. Displays Java version.
3. Lists the workspace.
4. Simulates a successful build.
